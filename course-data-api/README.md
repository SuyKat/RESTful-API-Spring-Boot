# Course APi
**Endpoints**

GET
- http://localhost:8080/topics
- http://localhost:8080/topics/{id}

POST
- http://localhost:8080/topics
  
    Provide one topic in body as json object

PUT

- http://localhost:8080/topics/{id}
  
  Provide updated info in the body for {id}

PUT

- http://localhost:8080/topics/{id}
   
   Deletes entry for {id}
